This is the open-source codebase behind **Idle Industries**, a factory-building sim with idle mechanics to allow you to set stuff up and come back later.

##  Getting Started

**Prerequisites**
- [Unity Hub](https://unity.com/download) with a Unity Editor version matching the one listed in [`ProjectSettings/ProjectVersion.txt`](./ProjectSettings/ProjectVersion.txt)
- Git

**Setup**

```bash
git clone https://github.com/maxtarabbia/IdleFactory.git
```

1. Open **Unity Hub** → **Add project from disk** → select the cloned `IdleFactory` folder.
2. Let Unity import assets and resolve packages — this can take a few minutes on the first open.
3. Open the main scene from `Assets/` and hit **Play** in the editor.
